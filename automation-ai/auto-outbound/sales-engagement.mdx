Connect Koala with your sales engagement platform to automatically trigger sequences, create tasks, and personalize outreach based on real-time intent signals.

## Outreach Integration

### Overview
Trigger Outreach sequences and create prospects automatically when high-intent visitors are detected.

### Key Features
- Auto-create prospects from anonymous visitors
- Trigger sequences based on intent signals
- Personalize emails with browsing data
- Track engagement back to Koala

### Setup & Configuration

<Steps>
  <Step title="Connect Outreach">
    1. Go to Settings → Integrations → Outreach
    2. Click "Connect Outreach"
    3. Authorize Koala to access your Outreach instance
  </Step>
  
  <Step title="Configure Sequences">
    Map intent signals to Outreach sequences:
    - Pricing page → "High Intent Pricing" sequence
    - Demo page → "Demo Interest" sequence
    - Docs/Integration → "Technical Buyer" sequence
  </Step>
  
  <Step title="Set Trigger Rules">
    Define when to create prospects:
    - Intent score threshold (e.g., >70)
    - Specific page visits
    - Company size/industry filters
  </Step>
</Steps>

### Personalization Variables
Use these Koala variables in your Outreach templates:
- `{{koala.pages_viewed}}` - List of pages visited
- `{{koala.intent_score}}` - Current intent score
- `{{koala.last_page}}` - Most recent page
- `{{koala.time_on_site}}` - Total time spent

## Salesloft Integration

### Overview
Automatically add people to Salesloft cadences and create tasks for your team based on buying signals.

### Automation Options
1. **Cadence Triggers**: Start cadences for new high-intent visitors
2. **Task Creation**: Create calls/emails for existing people
3. **Person Enrichment**: Update Salesloft records with intent data
4. **Account Alerts**: Notify reps of account-level activity

### Best Practices
- Use intent score for cadence prioritization
- Create separate cadences for different intent signals
- Include browsing context in first touch
- Set up round-robin for fair lead distribution

## Apollo Integration

### Overview
Enrich contacts with Apollo data and trigger automated sequences based on Koala intent signals.

### Unique Features
- **Contact Enrichment**: Find email/phone for anonymous visitors
- **Sequence Automation**: Trigger Apollo sequences
- **List Building**: Add high-intent visitors to Apollo lists
- **Data Sync**: Two-way sync of engagement data

### Setup Process

<Tabs>
  <Tab title="Initial Setup">
    1. Connect Apollo account in Settings
    2. Configure API permissions
    3. Set enrichment preferences
    4. Map custom fields
  </Tab>
  
  <Tab title="Automation Rules">
    Create rules for different scenarios:
    - New company → Enrich and add to sequence
    - Return visitor → Update engagement score
    - High intent → Create task for rep
  </Tab>
  
  <Tab title="Enrichment Settings">
    Configure what data to pull:
    - Contact details (email, phone, LinkedIn)
    - Company information
    - Technology stack
    - Funding data
  </Tab>
</Tabs>

## Implementation Best Practices

### 1. Start Simple
- Begin with one tool and one use case
- Test with a small segment first
- Measure response rates before scaling

### 2. Personalization Strategy
- Reference specific pages visited
- Mention time-relevant triggers
- Use industry-specific messaging
- Avoid creepy specificity

### 3. Timing Optimization
- Immediate response for pricing page visits
- 24-hour delay for content consumption
- Weekly batch for low-intent signals
- Respect timezone differences

### 4. Compliance Considerations
- Ensure GDPR/CCPA compliance
- Respect opt-out preferences
- Document data usage
- Maintain audit trails
