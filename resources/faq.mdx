## General Questions

<AccordionGroup>
  <Accordion title="What is Koala?">
    Koala is a customer intent platform that helps B2B companies identify, understand, and engage with prospects showing buying signals. We track visitor behavior across your website, docs, and product to surface high-intent leads for your sales team.
  </Accordion>
  
  <Accordion title="How is Koala different from other analytics tools?">
    Unlike traditional analytics tools that focus on aggregate metrics, Koala:
    - Identifies individual visitors and companies
    - Focuses on sales-relevant intent signals
    - Provides real-time alerts for immediate action
    - Integrates deeply with your sales stack
    - Includes built-in automation capabilities
  </Accordion>
  
  <Accordion title="What size companies use Koala?">
    Koala serves B2B companies from seed-stage startups to public companies. Our sweet spot is companies with:
    - 10-500 employees
    - B2B SaaS or services
    - Average deal size >$10k
    - Sales team of 5+ reps
  </Accordion>
  
  <Accordion title="How quickly can I get started?">
    Most teams are up and running within 15 minutes for basic tracking. Full implementation including integrations typically takes 2-5 days.
  </Accordion>
</AccordionGroup>

## Technical Questions

<AccordionGroup>
  <Accordion title="How does Koala identify anonymous visitors?">
    Koala uses multiple methods:
    1. **IP intelligence**: Partnership with Clearbit Reveal for company matching
    2. **Cross-domain tracking**: Unified profiles across your properties
    3. **Identity graph**: Connecting anonymous and known sessions
    4. **Third-party enrichment**: Additional data sources for accuracy
  </Accordion>
  
  <Accordion title="Will Koala slow down my website?">
    No. The Koala SDK:
    - Loads asynchronously (non-blocking)
    - Adds <15ms to page load time
    - Uses CDN for global performance
    - Batches requests efficiently
    - Has 99.99% uptime SLA
  </Accordion>
  
  <Accordion title="What data does Koala collect?">
    By default, Koala collects:
    - Page URLs and titles
    - Referrer information
    - Browser and device data
    - IP address (for company matching)
    - Form submissions (with email detection)
    - Custom events you define
    
    All data collection is configurable and GDPR-compliant.
  </Accordion>
  
  <Accordion title="Can Koala work with my existing analytics?">
    Yes! Koala complements your existing tools:
    - Import data from Segment, Mixpanel, etc.
    - Run alongside Google Analytics
    - Export to your data warehouse
    - No need to replace existing tracking
  </Accordion>
</AccordionGroup>

## Troubleshooting

<AccordionGroup>
  <Accordion title="I'm not seeing any data">
    Common causes and solutions:
    1. **Just installed**: Wait 5-10 minutes for data
    2. **Wrong API key**: Verify in Settings
    3. **Ad blocker**: Test in incognito mode
    4. **JavaScript errors**: Check browser console
    5. **Filters applied**: Remove all filters in dashboard
  </Accordion>
  
  <Accordion title="Visitors aren't being identified">
    To improve identification:
    1. Ensure Clearbit integration is active
    2. Implement server-side identify calls
    3. Track form submissions
    4. Connect your CRM for matching
    5. Use custom identify events
  </Accordion>
  
  <Accordion title="Slack alerts aren't working">
    Check these items:
    1. Slack integration is connected
    2. Alert rules are active
    3. Correct channel selected
    4. Bot has channel permissions
    5. No filters blocking alerts
  </Accordion>
  
  <Accordion title="Integration sync issues">
    For any integration issues:
    1. Re-authenticate the connection
    2. Check API limits/quotas
    3. Verify field mappings
    4. Review error logs
    5. Contact support if persists
  </Accordion>
</AccordionGroup>
